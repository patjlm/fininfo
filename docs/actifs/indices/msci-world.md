# MSCI World

> Indice de référence mondial couvrant les grandes et moyennes capitalisations de 23 pays développés. Représente environ 85 % de la capitalisation boursière flottante de chaque pays. Très utilisé comme benchmark pour les fonds actions monde développé.

## Caractéristiques
<!-- Mis à jour : 2026-05-17 -->

- **Fournisseur** : MSCI
- **Date de lancement** : 1986-03-31
- **Devise de référence** : USD
- **Nombre de constituants** : 1 310
- **Zone géographique** : 23 pays développés (Australie, Autriche, Belgique, Canada, Danemark, Finlande, France, Allemagne, Hong Kong, Irlande, Israël, Italie, Japon, Pays-Bas, Nouvelle-Zélande, Norvège, Portugal, Singapour, Espagne, Suède, Suisse, Royaume-Uni, États-Unis)
- **Capitalisation** : large et mid cap (~85 % de la capitalisation flottante par pays)
- **Méthode de pondération** : capitalisation flottante (free float-adjusted)
- **Fréquence de rebalancement** : trimestriel (février, mai, août, novembre)
- **Critères d'inclusion** : capitalisation boursière suffisante, liquidité minimale, flottant ajusté — couvre environ 85 % de la capitalisation flottante dans chaque pays

## Variantes

- **MSCI World SRI** : version filtrée selon des critères ESG stricts (Socially Responsible Investing)
- **MSCI World ESG Enhanced** : version optimisée ESG avec tracking error minimale
- **MSCI World Climate Paris Aligned** : aligné sur les objectifs de l'Accord de Paris
- **MSCI World Equal Weighted** : version équipondérée
- **MSCI World Momentum** : facteur momentum
- **MSCI World Quality** : facteur qualité
- **MSCI World Value** : facteur value
- **MSCI World Minimum Volatility** : optimisé pour la volatilité minimale
- **MSCI World hedged** : versions couvertes en devise (EUR, CHF, etc.)
- **MSCI World Small Cap** : voir [MSCI World Small Cap](msci-world-small-cap.md)
