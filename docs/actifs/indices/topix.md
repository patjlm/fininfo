# TOPIX (Tokyo Stock Price Index)

> Indice de référence du marché actions japonais, couvrant une proportion extensive des actions cotées à la Bourse de Tokyo. Utilisé comme benchmark pour les investissements en actions japonaises et les fonds de pension.

## Caractéristiques
<!-- Mis à jour : 2026-05-17 -->

- **Fournisseur** : JPX Market Innovation & Research, Inc. (Japan Exchange Group)
- **Début de calcul** : 1er juillet 1969
- **Date de base** : 4 janvier 1968 (base 100 points)
- **Devise de référence** : JPY
- **Nombre de constituants** : 1 650 (au 30/04/2026)
- **Zone géographique** : Japon (Tokyo Stock Exchange)
- **Capitalisation** : toutes capitalisations (large, mid, small)
- **Méthode de pondération** : capitalisation flottante (free-float adjusted)
- **Weight cap** : 10 %
- **Fréquence de rebalancement** : annuel (dernier jour ouvré d'octobre) — à partir d'octobre 2026
- **Fréquence de calcul** : temps réel (toutes les secondes) pour le Price Return ; uniquement cours de clôture pour le Total Return / Net Total Return
- **Critères d'inclusion** : actions listées au Tokyo Stock Exchange, avec application d'un facteur de liquidité ; les titres insuffisamment liquides voient leur pondération réduite progressivement

## Variantes
