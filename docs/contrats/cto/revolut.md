# Revolut — Compte-Titres

> Compte-titres proposé par [Revolut](https://www.revolut.com/fr-FR/) (Royaume-Uni / Lituanie). Néo-banque et application financière tout-en-un, avec plus de 50 millions de clients dans le monde. Services d'investissement fournis par Revolut Securities Europe UAB (Lituanie). Voir aussi la fiche [CTO (enveloppe)](../../enveloppes/cto.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Courtier** : Revolut Securities Europe UAB (Vilnius, Lituanie) — filiale de Revolut Ltd (Royaume-Uni)
- **Pays / régulation** : Lituanie — Banque de Lituanie (Lietuvos bankas) pour les services d'investissement ; Revolut France est une succursale de Revolut Bank UAB (SIREN 917 420 077)
- **Domiciliation du compte** : Lituanie (déclaration obligatoire comme compte étranger — formulaire 3916/3916-bis)
- **Pays disponibles** : 40
- **Dépôt initial minimum** : 1 EUR
- **CTO enfant** : non
- **Année de création** : 2015

## Produits et fonctionnalités
<!-- Mis à jour : 2026-05-26 -->

- **Actions** : + 3 000 (actions américaines et européennes, y compris ADR)
- **ETF** : ~ 158 (ETF non complexes enregistrés dans l'UE — indices, sectoriels, obligataires, matières premières)
- **Crypto** : oui — + 80 crypto-monnaies
- **Fractions d'actions/ETF** : oui — à partir de 1 EUR (minimum 0,00000001 part) ; pas de droits de vote sur les fractions
- **Investissement programmé (DCA)** : oui — gratuit sur ETF éligibles (sans commission même au-delà de l'allocation mensuelle), non disponible sur les actions ; auto top-up depuis le compte principal
- **Rémunération des liquidités** : oui — via Compte d'épargne ou Fonds monétaires flexibles (taux variables selon le plan)
- **Robo-Advisor** : oui — frais de 0,75 % par an ; portefeuille sur-mesure à partir d'actions US, EU et ETF
- **Plateforme de trading** : application mobile et web — interface intuitive, arrondis de petite monnaie, planificateur de budget
- **Note produits** : 3/5

## Propriété et transfert
<!-- Mis à jour : 2026-05-26 -->

- **Propriété des titres** : détention via nominee — Revolut détient le titre juridique (legal title), le client est bénéficiaire effectif (beneficial owner) avec droits absolus sur la valeur financière
- **Transfert de compte** : oui — frais de 35 USD par position transférée en cas de transfert sortant vers un autre courtier
- **IFU fourni** : non pour le trading (relevés fiscaux disponibles dans l'app pour aide à la déclaration manuelle) ; IFU fourni uniquement pour le Compte d'épargne
- **Déclaration compte étranger** : obligatoire — formulaire 3916/3916-bis (compte de courtage domicilié en Lituanie chez Revolut Securities Europe UAB) ; le compte bancaire principal est domicilié en France depuis 2024
- **Protection des investisseurs** : 22 000 EUR par client (régime d'assurance responsabilité envers les investisseurs de la République de Lituanie)
- **Note propriété** : 5/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-26 -->

- **Types d'ordres** : marché, limite, stop, stop-limit ; CFD : take-profit et stop-loss
- **Marchés accessibles** : actions US (NYSE, NASDAQ), actions européennes, ADR d'entreprises internationales cotées aux USA
- **CFD** : disponibles sur actions, indices et matières premières
- **Restrictions** : pas de PEA ni PEA-PME ; offre ETF limitée (~ 158) ; fractions d'actions sans droits de vote ; frais de change et frais week-end variables selon le plan
- **Particularités fiscales** : compte de courtage étranger (Lituanie) — déclaration annuelle obligatoire (formulaire 3916/3916-bis) ; pas de PFU prélevé à la source sur les plus-values, le client doit déclarer via formulaires 2042, 2074, 2086 (crypto)
- **Limites de trading** : ordres d'achat jusqu'à 100 000 USD (ou 50 000 parts)
