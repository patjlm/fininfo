# Yomoni Retraite+

> PER assurantiel distribué par Yomoni, assuré par Spirica. Contrat d'assurance-vie de groupe multisupport souscrit par l'Association Retraite Falguière.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Yomoni
- **Assureur** : Spirica — filiale du Crédit Agricole Assurances (solvabilité : 151,6 %)
- **Type** : Assurantiel
- **Modes de gestion** : gestion pilotée uniquement (pas de gestion libre)
- **Disponibilité** : 100 % en ligne
- **Versement minimum** : 1 000 EUR
- **Année de création** : 2023 (lancé le 9 octobre 2023)
- **Âge** : ouvert de 18 à 67 ans, versements possibles jusqu'à 77 ans
- **Frais d'adhésion** : 10 EUR (association Retraite Falguière)

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,30 % (Spirica) — inclus dans le tout compris
- **Gestion fonds euros** : 2 % maximum
- **Arbitrage** : 0 %
- **Rente (encours)** : 0,80 % / an sur les encours de rente
- **Arrérage de rente** : 0 %
- **Surcoût gestion pilotée** : 0,70 % / an (frais Yomoni)
- **Frais tout compris enveloppe ETF** : 1,60 % / an (0,30 % Spirica UC + 0,70 % Yomoni + ~0,30 % à 0,60 % ETF intégrés)
- **Frais tout compris enveloppe multi-actifs** : jusqu'à 2,20 % / an
- **Frais de sortie en capital** : 0 %
- **Frais de transfert sortant** : 1 % si < 5 ans, 0 % après 5 ans
- **Note frais** : 3/5

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponible** : Fonds Euro PER Nouvelle Génération (garantie 98 % — 100 % max du contrat)
- **Performance 2024** : 3,13 %
- **Performance 2025** : 3,08 %
- **Performance cumulée 5 ans** : 13,29 %
- **Plafond** : 5 millions EUR tous contrats Spirica confondus
- **Contraintes d'accès** : allocation déterminée par le profil de gestion pilotée choisi (pas de choix direct par l'épargnant)
- **Composition (31/12/2024)** : obligations 56 %, immobilier 32 %, monétaire 7 %, non coté 3 %, actions 2 %
- **Note fonds euros** : 5/5

## Unités de compte
<!-- Mis à jour : 2026-05-16 -->

- **Nombre total d'UC** : choix restreint (gestion pilotée uniquement)
- **ETF** : Yomoni Monde ISR, Yomoni Allocation ISR (gestion pilotée)
- **SCPI** : non disponible directement
- **Immobilier** : SC Y Immo (société civile immobilière, gérée par Épopée Gestion, anciennement Keys REIM)
- **Private Equity** : FCPR APEO (labellisé Relance)
- **Titres vifs** : non
- **Rééquilibrage automatique** : oui
- **Autres supports** : fonds ISR (Investissement Socialement Responsable)
- **Note UC** : 3,5/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Gestion pilotée obligatoire** : pas de gestion libre — l'épargnant ne peut pas choisir ses supports individuellement
- **Versement initial minimum** : 1 000 EUR
- **Versements libres minimum** : 250 EUR
- **Versements programmés minimum** : 100 EUR / mois
- **Arbitrage minimum** : 500 EUR (pour passer d'un profil de gestion pilotée à horizon vers la gestion libre) [à vérifier]
- **Arbitrage entre profils** : gratuit, sans frais
- **Choix de personnalisation** :
  - Avec ou sans désensibilisation progressive à l'approche de la retraite
  - Enveloppe ETF ou multi-actifs
  - Profil prudent, équilibré ou dynamique
- **Désensibilisation progressive** : activée par défaut, désactivable en un clic à la souscription — réduit progressivement le risque à l'approche de la retraite
- **Allocation au fonds euros** : déterminée par le profil de risque choisi, pas de choix direct
- **Sortie** : capital, rente ou panachage capital/rente
- **Options de rente** : annuités garanties, rente à paliers croissants/décroissants, réversion
- **Transfert entrant** : 0 % de frais — promotion : remboursement 100 % des frais de transfert avec code TRANSFERTPER jusqu'au 30 juin 2026
- **Transfert sortant** : 1 % si < 5 ans, gratuit après 5 ans (max légal)

### Différences avec Yomoni Retraite (PER bancaire)

- **Yomoni Retraite** (bancaire, lancé 2020) : 100 % ETF, pas de fonds euros (part sécurisée en fonds obligataire), frais d'enveloppe 0,30 %, conservé au Crédit Agricole Titres, garantie FGDR 100 000 EUR, droits de succession classiques
- **Yomoni Retraite+** (assurantiel, lancé 2023) : multi-actifs (ETF + fonds euros + immobilier + Private Equity), frais d'enveloppe 0,30 % (Spirica UC), assuré par Spirica, garantie FGAP 70 000 EUR, avantages successoraux de l'assurance-vie
