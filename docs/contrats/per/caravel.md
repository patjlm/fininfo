# Caravel PER

> PER assurantiel distribué par [Caravel](https://www.getcaravel.fr/), assuré par Apicil.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Caravel (fintech rachetée par Indexa Group en 2024, ORIAS 22002796)
- **Assureur** : Apicil Épargne Retraite (solvabilité : 218 %)
- **Type** : Assurantiel
- **Modes de gestion** : gestion déléguée (pilotée) — gestion libre non mentionnée sur le site officiel [à vérifier]
- **Disponibilité** : en ligne
- **Versement initial minimum** : 500 EUR
- **Année de création** : 2022

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,60 %
- **Gestion fonds euros** : 0,72 %
- **Arbitrage** : 0 %
- **Rente (arrérage)** : 1,50 %
- **Surcoût gestion pilotée** : 0,50 % selon sinvestir — le site officiel indique 1,38 % max tout compris (frais inclus) [à vérifier]
- **Frais des supports financiers (ETF)** : 0,16 % à 0,18 %
- **Frais associatifs (GERP)** : 8 EUR / an
- **Frais tout compris** : 1,36 % à 1,38 % / an selon le profil (annoncé par Caravel)
- **Options payantes (facultatives)** : Stop Loss 0,20 % des sommes arbitrées (min 100 EUR) ; Écrêtage des plus-values 0,20 % des sommes arbitrées (min 100 EUR) ; Lissage des investissements 0 %
- **Transfert entrant** : 0 %
- **Transfert sortant** : 1 % si contrat < 5 ans, 0 % après
- **Retrait anticipé** : 0 %
- **Note frais** : 4,5/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 500 EUR
- **Versements libres minimum** : 150 EUR
- **Versements programmés minimum** : 50 EUR / mois (modifiables ou suspendus sans frais à tout moment)
- **Arbitrage minimum** : non applicable (gestion pilotée, arbitrages gérés par Caravel)
- **Pas de gestion libre** : la composition du portefeuille est déléguée à Caravel selon le profil horizon retraite [à vérifier — sinvestir mentionne « gestion libre & pilotée »]
- **Règles d'arbitrage** : arbitrages automatiques réalisés par Caravel selon le profil horizon retraite
- **Limites d'investissement** : 50 % max en fonds euros (obligation de 50 % minimum en UC)
- **Conditions gestion pilotée** :
  - 3 phases automatiques basées sur l'âge :
    - **Croissance** (18-50 ans) : 100 % ETF actions, objectif 9 % / an
    - **Équilibre** (50-58 ans) : mix ETF actions/obligations, objectif 6 % / an
    - **Sécurisation** (58 ans+) : fonds monétaire + obligations + ETF, objectif 3 % / an
  - Portefeuilles composés principalement d'ETF ESG (alignés accords de Paris, exclusion énergies fossiles)
  - Surcoût : 0,50 % selon sinvestir (le site officiel annonce 1,38 % max tout compris sans détailler de surcoût séparé)
- **Transfert entrant** : gratuit, Caravel gère les démarches administratives (PERP, PERCO, Madelin, art. 83, autre PER) ; délai 1 à 2 mois
- **Transfert sortant** : 1 % si contrat < 5 ans, gratuit après 5 ans
- **Sortie** : en capital (0 % de frais) ou en rente viagère (1,50 % d'arrérage)
- **Délai de renonciation** : 30 jours après ouverture du contrat

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 4,06/5
- **Points forts** :
  - Frais parmi les plus bas du marché (1,36-1,38 % tout compris annoncé par Caravel)
  - 0 % de frais d'entrée, versement, arbitrage, transfert entrant et retrait
  - Large choix d'UC selon sinvestir : 100 ETF, 36 SCPI avec 100 % des loyers redistribués
  - Gestion pilotée automatique adaptée à l'horizon retraite avec 3 phases (Croissance / Équilibre / Sécurisation)
  - Approche ESG / investissement responsable (alignement accords de Paris, exclusion énergies fossiles)
  - Transfert entrant géré par Caravel
- **Points faibles** :
  - Pas de gestion libre (impossible de choisir ses supports individuellement)
  - Fonds euros moins performant que la moyenne (1,30 % en 2024)
  - Peu d'ancienneté du distributeur (créé en 2019, PER lancé en 2022)
  - Fonds euros progressivement remplacé par fonds monétaire (non garanti à 100 %)
  - Pas d'application mobile
- **Services complémentaires** : simulateur fiscal en ligne ; assurance-vie prévue pour le printemps 2026
