# Individuel Carac

> PER individuel assurantiel distribué et assuré par la Carac (mutuelle d'épargne fondée en 1924).
> Règles générales du PER : voir [enveloppe PER](../../enveloppes/per.md).

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Carac
- **Assureur** : Carac (solvabilité : 308,9 %)
- **Type** : Assurantiel (contrat mutualiste, adhésion au GERP Carac obligatoire)
- **Modes de gestion** : libre, profilée à horizon retraite
- **Disponibilité** : en ligne et en agence
- **Versement minimum** : 200 EUR
- **Année de création** : 2020
- **Éligibilité** : majeur, résident fiscal en France, moins de 71 ans
- **Souscription** : dématérialisée avec signature électronique, accompagnée par un conseiller Carac

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion fonds euros** : 0,60 % / an
- **Gestion UC** : 0,90 % / an (1 % sur certains supports) [à vérifier la distinction exacte par support]
- **Arbitrage** : 12 arbitrages gratuits / an, puis 1 % du montant arbitré
- **Rente (arrérages)** : 0 % [à vérifier — sinvestir indique 0,50 %, fiche tarifaire officielle non accessible en ligne]
- **Surcoût gestion profilée à horizon** : 0 %
- **Frais GERP Carac** : 0,15 % / an max des actifs du plan
- **Droits d'adhésion GERP Carac** : 20 EUR (une fois)
- **Note frais** : 2,5/5

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponibles** : Fonds euros Sécurité (garanti à 100 % -- 50 % max du contrat)
- **Performance 2025** : 4,05 %
- **Performance 2024** : 4,00 %
- **Performance cumulée 5 ans** : 14,15 % (période 2022-2025)
- **Contraintes d'accès** : 50 % min en UC obligatoire (impossible de placer 100 % en fonds euros)
- **Objectif 2026** : hypothèse de rendement 3,50 % + bonus de 1 % à 1,50 % [à vérifier conditions du bonus]
- **Note fonds euros** : 4/5

- **Nombre total d'UC** : 43 (gamme restreinte, dont 2 produits structurés à souscription temporaire)
- **ETF** : 0 (4 trackers indiciels BNP Paribas Easy en version OPCVM "Track Classic", non cotés en bourse)
- **SCPI** : 0

- **Limite supports immobiliers et private equity** : 30 % max des versements
- **Sociétés de gestion** : CPR AM, EdR AM, Thematics AM, Mirova, Dorval, Comgest, AllianceBernstein, Carmignac, Franklin Templeton, Lazard, Candriam, Tikehau, Ecofi, ATREAM, Eurazeo, Eiffel IG, BNP Paribas AM, Oddo BHF AM, Pictet AM, JPMorgan AM, Fidelity, Vontobel AM
- **Note UC** : 0/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

### Versements

- **Versement initial minimum** : 200 EUR
- **Versements libres minimum** : 200 EUR
- **Versements programmés minimum** : 50 EUR / mois, 150 EUR / trimestre, 300 EUR / semestre, 600 EUR / an
- **Arrêt des versements programmés** : possible à tout moment

### Arbitrages

- **Arbitrage minimum** : Non communiqué
- **Arbitrages gratuits** : 12 par année civile
- **Au-delà** : 1 % du montant arbitré
- **Changement de profil horizon** : 1 fois / an, sans frais
- **Changement de mode de gestion** : possible en cours de vie du contrat (libre vers horizon et inversement), sans frais
- **Arbitrages automatiques en gestion horizon** : semestriels, sans frais
- **Contrainte** : si le cumul UC dépasse 30 % [à vérifier le sens exact de cette règle -- potentiellement lié à la gestion horizon]

### Contraintes fonds euros

- **50 % max du contrat** en fonds euros
- **50 % min en UC** obligatoire pour accéder au fonds euros
- **100 % fonds euros impossible**

### Gestion profilée à horizon retraite

- **Profils disponibles** : prudent, équilibré, dynamique, offensif
- **Profil par défaut** : équilibré (loi PACTE)
- **Frais supplémentaires** : 0 % (inclus dans les frais de gestion)
- **Principe** : sécurisation progressive automatique à l'approche de la retraite
- **Profil prudent** : exposition max 35 % marchés boursiers, 15 % immobilier
- **Intègre du non coté** : oui (conformément à la loi Industrie Verte)
- **Performance 2024** :
  - Prudent : +3,77 %
  - Équilibré : +4,26 %
  - Dynamique : +5,59 %
- **Performance 2023** :
  - Prudent : +6,19 %
  - Équilibré : +7,36 %
  - Dynamique : +11,07 %
- **Performance cumulée 2023-2024** :
  - Prudent : +10,19 %
  - Équilibré : +11,93 %
  - Dynamique : +17,27 %

### Options de gestion automatique

- Sécurisation des plus-values
- Dynamisation des intérêts

### Transfert

- **Transfert entrant** : accepté depuis PER, PERP, Madelin, PERCO, art. 83 et autres contrats d'épargne-retraite ; 0 % de frais ; pas de montant minimum
- **Transfert sortant** : 1 % de l'encours transféré si contrat < 5 ans ; 0 % si contrat >= 5 ans

### Sortie

- **Modes de sortie** : capital, rente, ou combinaison des deux
- **Frais de sortie anticipée** : 0 %
- **Frais sur arrérages de rente** : 0 %
- **Garantie de table de mortalité** : non

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 2,92/5
- **Points forts** :
  - Fonds euros très performant (4,05 % en 2025, parmi les meilleurs du marché)
  - 0 % de frais sur versement (rare pour un acteur traditionnel avec conseillers)
  - Gestion profilée à horizon sans surcoût (0 % de frais de mandat)
  - 0 % de frais sur arrérages de rente
  - Accessible dès 200 EUR
  - Souscription en ligne ou en agence avec accompagnement
  - Solvabilité très élevée (308,9 %)
  - Récompense : Meilleurs PER 2026 (Challenges)
  - Sélection Premium Good Value for Money 2026 (4e année consécutive)
- **Points faibles** :
  - Gamme d'UC très limitée (43 supports dont 2 produits structurés temporaires)
  - Aucun ETF coté disponible (4 trackers indiciels BNP Paribas Easy en version OPCVM non cotée)
  - Aucune SCPI disponible
  - 50 % min en UC obligatoire (impossible d'être 100 % fonds euros)
  - Frais de gestion UC relativement élevés (0,90 % à 1 %)
  - Pas de rééquilibrage automatique en gestion libre
  - Pas d'application mobile
  - Avis clients faibles (2,2/5)
- **Services complémentaires** : assurance-vie (Carac Épargne Patrimoine, Carac Épargne Génération)
- **Fiabilité** : 4/5
