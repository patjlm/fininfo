# Multisupport Afer

> Assurance vie groupée distribuée par l'Afer (Association Française d'Épargne et de Retraite), assurée par Abeille Assurances (filiale de Aéma).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Afer (Association Française d'Épargne et de Retraite) — également distribué via Placement Direct
- **Assureur** : Abeille Assurances (filiale de Aéma) — solvabilité : Non communiqué [à vérifier sur afer.fr]
- **Type** : Multisupports (fonds garanti en euros + unités de compte + Afer Eurocroissance)
- **Modes de gestion** : Libre, conseillée (4 profils d'allocation : Prudent, Équilibré, Dynamique, Offensif)
- **Disponibilité** : En ligne et en agence (distribution directe par l'association Afer)
- **Versement minimum** : 100 EUR
- **Année de création** : 1976
- **Type de contrat** : Groupé (contrat collectif souscrit par l'association Afer)
- **Encours** : 37,02 milliards EUR (au 30 juin 2025, source : placement-direct.fr)
- **Nombre de membres** : environ 760 000 (source : placement-direct.fr)

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0,50 % sur les versements sur le Fonds Garanti en euros ; 0 % sur les versements en UC et Afer Eurocroissance (source : sinvestir.fr — à vérifier sur afer.fr)
- **Gestion UC** : 0,475 % / an (source : sinvestir.fr)
- **Gestion fonds euros** : 0,475 % / an (source : sinvestir.fr)
- **Gestion Afer Eurocroissance** : 0,89 % / an (source : sinvestir.fr)
- **Arbitrage** : 0 % (source : sinvestir.fr)
- **Surcoût gestion sous mandat** : +0,25 % / an (source : sinvestir.fr)
- **Frais de rachat** : 0 %
- **Autres frais** :
  - 20 EUR de frais d'adhésion à l'association (paiement unique à l'ouverture)
  - 0,055 % / an de frais de garantie plancher (décès avant 75 ans)
  - 3 % de frais sur les versements de rente
  - Taux avance 2023 : 2,66 %
- **Note frais** : 4,4/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponibles** :
  - Fonds Garanti en euros (garanti à 98 % — 75 % maximum du contrat)
  - Afer Eurocroissance (support euro-croissance, garantie partielle à terme — frais 0,89 %)
- **Performance 2024** : 2,51 % (Fonds Garanti en euros, net de frais de gestion, hors prélèvements fiscaux et sociaux — source : sinvestir.fr)
- **Performance 2025** : 2,65 % (Fonds Garanti en euros, net de frais de gestion, hors prélèvements fiscaux et sociaux — source : placement-direct.fr)
- **Performance cumulée 5 ans** : 11,09 % (source : sinvestir.fr)
- **Contraintes d'accès** : maximum 75 % du contrat en Fonds Garanti en euros (source : sinvestir.fr — à vérifier sur afer.fr)
- **Note fonds euros** : 4/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 100 EUR (source : placement-direct.fr)
- **Versements libres minimum** : 100 EUR (source : placement-direct.fr)
- **Versements programmés minimum** : 50 EUR / mois (source : placement-direct.fr)
- **Arbitrage minimum** : Non communiqué [à vérifier sur afer.fr]
- **Règles d'arbitrage** : 0 % de frais d'arbitrage ; arbitrages programmés disponibles (source : placement-direct.fr)
- **Limites d'investissement** : 75 % maximum en Fonds Garanti en euros (source : sinvestir.fr — à vérifier)
- **Conditions gestion conseillée** : 4 profils d'allocation suggérés — Prudent (70 % fonds euros, 30 % actifs risqués), Équilibré (50/50), Dynamique (20 % fonds euros, 80 % actifs risqués), Offensif (100 % UC) ; surcoût +0,25 % / an sous mandat (source : placement-direct.fr et sinvestir.fr)
- **Outils d'automatisation** :
  - Arbitrages programmés (transferts automatiques vers les UC sélectionnées)
  - Sécurisation des plus-values (transferts automatiques vers le fonds euros quand seuil atteint)
  - Dynamisation des intérêts (réinvestissement automatique des intérêts du fonds euros en UC)
- **Rachat partiel minimum** : Non communiqué [à vérifier sur afer.fr]
- **Avance sur contrat** : oui (taux 2023 : 2,66 % — à vérifier sur afer.fr pour le taux 2025/2026)
- **Clause bénéficiaire** : Non communiqué [à vérifier sur afer.fr]
- **Garantie plancher** : oui — 0,055 % / an ; capital garanti aux bénéficiaires en cas de décès avant 75 ans

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)
