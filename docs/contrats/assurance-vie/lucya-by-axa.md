# Lucya by AXA

> Assurance vie distribuée par Lucya (assurancevie.com), assurée par AXA.

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Lucya (anciennement assurancevie.com — ~170 000 clients, 6 Md€ d'AUM)
- **Assureur** : AXA (solvabilité : Non communiqué)
- **Type** : Multisupports groupé
- **Modes de gestion** : Libre, gestion déléguée Lucya (3 profils), gestion déléguée Sélection LIVE AXA IM (3 niveaux)
- **Disponibilité** : En ligne exclusivement
- **Versement minimum** : 1 000 EUR (gestion libre) / 1 000 EUR (gestion déléguée, versement libre dès 500 EUR)
- **Année de création** : 2001

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,50 % / an
- **Gestion fonds euros** : 0,70 % / an (Fonds Général et Fonds Croissance)
- **Arbitrage** : 0 % (illimités et gratuits) — 0,10 % sur les ETF
- **Surcoût gestion déléguée** : +0,25 % / an sur les UC
- **Frais de rachat** : 0 %
- **Autres frais** : 10 % de frais de performance sur le Fonds Croissance
- **Note frais** : 1/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds Général Lucya by AXA** : garantie permanente du capital — 100 % possible
  - Performance 2024 : Non communiquée
  - Performance 2025 (net de frais de gestion, hors PS et fiscalité) :
    - 2,40 % si part d'UC ≥ 30 % et < 40 %
    - 2,85 % si part d'UC ≥ 40 % et < 50 %
    - 3,40 % si part d'UC > 50 %
  - Performance cumulée 5 ans : Non communiquée
- **Fonds Croissance** : garantie 100 % du capital net de frais de gestion à échéance 10 ans (eurocroissance) — combinaison obligations + actifs diversifiés (infrastructure, private equity, immobilier, forêts)
  - Performance 2024 : Non communiquée
  - Performance 2025 (net de frais de gestion, hors PS et fiscalité) :
    - 2,65 % si part d'UC ≥ 30 % et < 40 %
    - 3,15 % si part d'UC ≥ 40 % et < 60 %
    - 3,75 % si part d'UC > 50 %
  - Performance cumulée 5 ans : Non communiquée
- **Contraintes d'accès** : aucune contrainte minimale d'UC pour accéder au fonds euros — les taux sont bonifiés selon la part d'UC détenue
- **Note fonds euros** : 4/5 (source : sinvestir.fr)

## Unités de compte
<!-- Mis à jour : 2026-05-16 -->

- **Nombre total d'UC** : ~850
- **ETF** : 25 (couvrant les principaux indices boursiers)
- **SCPI** : 0 (aucune SCPI disponible)
- **Titres vifs** : non
- **Rééquilibrage automatique** : non
- **Autres supports** : private equity (FCPR), immobilier (OPCI), fonds obligataires, produits structurés, fonds ISR (labels ISR, Greenfin, Finansol, Relance)
- **Note UC** : 4/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 1 000 EUR
- **Versements libres minimum** : 500 EUR
- **Versements programmés minimum** : 50 EUR / mois — 150 EUR / trimestre — 300 EUR / semestre — 600 EUR / an
- **Arbitrage minimum** : Non communiqué
- **Règles d'arbitrage** : arbitrages illimités et gratuits (hors ETF à 0,10 %) ; aucune contrainte spécifique d'arbitrage vers les fonds euros mentionnée
- **Limites d'investissement** : 100 % fonds euros possible ; aucune SCPI disponible
- **Conditions gestion déléguée** :
  - Gestion déléguée Lucya : 3 profils (Modérée, Équilibrée, Dynamique) — horizon recommandé 3, 5 et 8 ans respectivement — surcoût +0,25 % / an — performances 2024 : Modérée +9,66 %, Équilibrée +9,94 %, Dynamique +10,95 % — performances 2025 (au 10/04/2026) : Modérée +3,13 %, Équilibrée +5,10 %, Dynamique +7,74 %
  - Gestion déléguée Sélection LIVE (AXA IM Select) : 3 niveaux (LIVE 2, LIVE 5, LIVE 8) — conditions et performances non communiquées
  - Les modes de gestion sont combinables au sein d'un même contrat
- **Changement de profil / mode de gestion** : possible à tout moment, sans frais
- **Rachat partiel minimum** : Non communiqué
- **Avance sur contrat** : disponible (pour financer un besoin momentané, avec intérêts) — conditions précises non communiquées sur le site
- **Clause bénéficiaire** : standard, libre, nominative ou notariée

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)
