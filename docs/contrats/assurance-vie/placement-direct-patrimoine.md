# Placement-direct Patrimoine

> Assurance vie distribuée par Placement Direct, assurée par Garance (mutuelle soumise au Code de la mutualité).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Placement Direct
- **Assureur** : Garance (mutuelle soumise aux dispositions du Livre II du Code de la mutualité — SIREN 391 399 227 — siège : 51, Rue de Châteaudun, 75442 Paris Cedex 09)
- **Type** : Multisupports (fonds euros + UC), branches 20 et 22 du Code de la mutualité (art. R 211-2)
- **Modes de gestion** : Libre, pilotée, profilée à horizon (prudent / équilibré / dynamique)
- **Disponibilité** : En ligne exclusivement (indisponible pour les mineurs, les majeurs protégés et les non-résidents français)
- **Versement minimum** : 500 EUR (ou 50 EUR avec versements programmés)
- **Année de création** : 1999 (contrat récent — données de performance limitées)

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,50 % / an
- **Gestion fonds euros** : 0,60 % / an
- **Arbitrage** : 0 %
- **Changement de mode de gestion** : 0 %
- **Surcoût gestion pilotée / profilée** : +0,25 % / an
- **Frais de rachat** : 0 %
- **Autres frais** : 3 % sur les versements en cas de sortie en rente
- **Note frais** : 4,9/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponible** : Actif général de Garance (garanti à 100 % — 100 % du contrat possible)
- **Performance 2024** : Non communiqué (contrat récent, pas de données disponibles pour 2024)
- **Performance 2025** :
  - 2,80 % si part UC < 40 %
  - 3,55 % si 40 % ≤ UC < 60 %
  - 4,70 % si part UC ≥ 60 %
- **Performance cumulée 5 ans** : Non disponible (contrat récent — données 2025 uniquement)
- **Contraintes d'accès** : Rendement du fonds euros bonifié selon la part allouée aux UC ; aucune limite d'allocation (100 % fonds euros possible, mais rendement minimal si < 40 % UC)
- **Note fonds euros** : 5/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 500 EUR (ou 50 EUR avec mise en place de versements programmés)
- **Versements libres minimum** : 50 EUR
- **Versements programmés minimum** : 50 EUR / mois ; 150 EUR / trimestre ; 300 EUR / semestre ; 600 EUR / an
- **Arbitrage minimum** : 50 EUR
- **Règles d'arbitrage** : 0 % de frais ; libre entre tous les supports
- **Limites d'investissement** : 100 % fonds euros possible ; rendement du fonds euros bonifié selon la part d'UC (paliers à 40 % et 60 % d'UC)
- **SCPI** : 4 SCPI sans frais d'entrée, sans limite maximale d'investissement, sans pénalités de retrait
- **Conditions gestion pilotée** : surcoût +0,25 % / an ; profils disponibles : prudent, équilibré, dynamique (horizon de placement)
- **Conditions gestion profilée à horizon** : allocation auto-ajustée selon la durée restante avant l'échéance choisie
- **Rachat partiel minimum** : Non communiqué [à vérifier]
- **Avance sur contrat** : Non communiqué [à vérifier]
- **Clause bénéficiaire** : Non communiqué [à vérifier]
- **Souscription non disponible pour** : mineurs, majeurs protégés, non-résidents français

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 4,65/5 (source : sinvestir.fr)
- **Points forts** :
  - Fonds euros très performant en 2025 (jusqu'à 4,70 % si UC ≥ 60 %) sans contrainte de quota minimum d'UC obligatoire
  - 4 SCPI sans frais d'entrée, sans limite d'investissement, sans pénalités de retrait
  - Frais compétitifs : 0 % versement, 0 % arbitrage, 0,50 % UC / an
  - Surcoût gestion pilotée faible (+0,25 % / an — parmi les plus bas du marché)
  - Versements libres et arbitrages dès 50 EUR
  - Versements programmés flexibles (mensuel, trimestriel, semestriel, annuel)
  - Rééquilibrage automatique disponible
  - 100 % des loyers SCPI reversés
- **Points faibles** :
  - Offre UC limitée (94 supports vs plus de 1 200 pour Placement-direct Vie)
  - Absence de titres vifs
  - Peu d'historique de performance (contrat récent — pas de données 2024 sur le fonds euros)
  - Assureur Garance peu connu, données de solvabilité non publiées sur le site distributeur
  - Rachat partiel minimum, avance sur contrat et clause bénéficiaire non documentés en ligne
- **Services complémentaires** : PER (Placement-direct PER), compte à terme
