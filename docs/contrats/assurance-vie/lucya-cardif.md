# Lucya Cardif

> Assurance vie distribuée par Lucya, assurée par BNP Paribas Cardif (filiale de BNP Paribas).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Distributeur** : Lucya (plateforme en ligne, CIF réglementé AMF)
- **Assureur** : BNP Paribas Cardif (Cardif Assurance Vie — filiale de BNP Paribas — CA 19,1 Md€ en 2025 — UC 72,4 Md€ + fonds général 101,4 Md€ — ratio SCR : 170 % au 31/12/2025)
- **Type** : Multisupports groupé
- **Modes de gestion** : Libre, déléguée (3 profils)
- **Disponibilité** : En ligne exclusivement (souscription sur assurancevie.com)
- **Versement minimum** : 500 EUR (gestion libre) / 1 000 EUR (gestion déléguée)
- **Année de création** : 2001

## Frais
<!-- Mis à jour : 2026-05-26 -->

- **Versement** : 0 %
- **Gestion UC** : 0,50 % / an
- **Gestion fonds euros (Fonds général)** : 0,70 % / an
- **Gestion fonds euros (Fonds Euro Private Strategies)** : 3 % / an maximum
- **Arbitrage** : 0 % (illimité, sans minimum)
- **Arbitrage ETF** : 0,10 % par transaction
- **Surcoût gestion déléguée** : +0,25 % / an (maximum 0,50 %)
- **Frais de rachat** : 0 %
- **Frais d'adhésion** : 10 EUR (paiement unique à l'ouverture, association souscripteurs)
- **Pénalités SCPI** : 3 % sur les sommes rachetées ou arbitrées sur les supports SCPI dans les 3 ans suivant l'investissement sur ce support
- **Frais de rente** : 2,75 % sur les versements de rente
- **Note frais** : 4,5/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-26 -->

- **Fonds Euro Général** (garanti à 99,3 % — 100 % possible)
  - Performance 2024 : 2,75 %
  - Performance 2025 : 2,75 %
  - Performance cumulée 5 ans : 11,60 %
  - Contraintes d'accès : aucune contrainte particulière ; 100 % fonds euros possible sans condition d'UC minimale
  - Bonus 2026-2027 : jusqu'à +1,50 % (hypothèse de taux 4,25 %) — conditions : minimum 10 000 EUR versés et 45 % de l'encours en UC — non garanti (source : lucya.com/rendement-des-fonds-euros-2025/)
- **Fonds Euro Private Strategies** (garanti à 97 % — actifs non cotés)
  - Performance 2024 : 3,00 %
  - Performance 2025 : 2,75 %
  - Performance cumulée 5 ans : 14,27 %
  - Contraintes d'accès : accessible uniquement par versements libres (non programmés) — montant minimum 500 EUR — le montant investi en UC doit être au moins 2 fois supérieur au montant investi dans ce fonds (ce fonds ne peut donc pas dépasser 1/3 d'un versement) — ne peut pas représenter plus de 1/3 de l'encours total du contrat
  - Frais de gestion : 3 % / an (inclus dans la performance nette affichée)
- **Note fonds euros** : 4,25/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-26 -->

- **Versement initial minimum** : 500 EUR (gestion libre) / 1 000 EUR (gestion déléguée)
- **Versements libres minimum** : 500 EUR
- **Versements programmés minimum** : 50 EUR / mois — 150 EUR / trimestre — 300 EUR / semestre — 600 EUR / an
- **Arbitrage minimum** : pas de montant minimum
- **Rachat partiel minimum** : 100 EUR (ponctuel) — 100 EUR / mois (programmé)
- **Règles d'arbitrage** :
  - Arbitrage gratuit et illimité entre tous supports (sans minimum)
  - ETF : 0,10 % de frais par opération d'arbitrage (achat, vente ou arbitrage)
  - SCPI : pénalité de 3 % sur les sommes arbitrées dans les 3 ans suivant l'investissement sur ce support
- **Limites d'investissement** :
  - Fonds Euro Général : 100 % possible, sans contrainte
  - Fonds Euro Private Strategies : ne peut pas dépasser 1/3 du contrat ; l'investissement en UC au moment du versement doit être au moins 2x le montant investi dans ce fonds ; accessible uniquement par versements libres (non programmés)
- **Conditions gestion déléguée** :
  - Minimum : 1 000 EUR
  - 3 profils disponibles : Modéré (horizon 3 ans, risque faible), Équilibré (horizon 5 ans), Dynamique (horizon 8 ans, 40-100 % actions)
  - Frais supplémentaires : +0,25 % / an (maximum 0,50 %)
  - Changement de profil ou de mode de gestion : gratuit, à tout moment
  - Performances 2024 : Modéré +9,66 % — Équilibré +9,94 % — Dynamique +10,95 %
  - Performances 2025 : Modéré +3,13 % — Équilibré +5,10 % — Dynamique +7,74 %
  - Performance depuis origine (Modéré, depuis 06/2023) : +7,92 % cumulé (à mars 2026)
- **Avance sur contrat** : disponible (sous conditions de l'assureur)
- **Clause bénéficiaire** : standard / libre / démembrée
