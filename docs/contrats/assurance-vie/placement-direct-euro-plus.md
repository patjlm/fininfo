# Placement-direct Euro+

> Assurance vie distribuée par Placement Direct, assurée par SwissLife Assurance et Patrimoine. Contrat collectif souscrit via l'association AGIS.

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Distributeur** : Placement Direct
- **Assureur** : SwissLife Assurance et Patrimoine
- **Type** : Multisupports (fonds euros nouvelle génération + UC), contrat collectif
- **Association souscriptrice** : AGIS (adhésion unique obligatoire à l'ouverture : 10 EUR)
- **Modes de gestion** : Libre (allocation libre) ; Déléguée (allocation déléguée par SwissLife Banque Privée)
- **Disponibilité** : En ligne exclusivement
- **Versement minimum** : 500 EUR
- **Année de création** : 2026 (contrat récent — données de performance du fonds euros disponibles depuis 2023)

## Frais
<!-- Mis à jour : 2026-05-26 -->

- **Versement** : 0 %
- **Gestion UC** : 0,60 % / an
- **Gestion fonds euros** : 0,60 % / an
- **Arbitrage** : 0 % (options d'arbitrage automatiques gratuites)
- **Surcoût gestion déléguée** : +0,40 % / an
- **Frais de rachat** : 0 %
- **Autres frais** :

  - Frais sur encours après décès : 0,60 % / an
- **Note frais** : 4,25/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-26 -->

- **Fonds disponible** : SwissLife Euro+ (garanti à 99,4 % avant frais de gestion)
- **Performance 2023** : 4,10 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance 2024** : 3,60 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance 2025** : 3,60 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance cumulée 3 ans (2023–2025)** : 11,30 % (source : sinvestir.fr)
- **Contraintes d'accès** :
  - Minimum 30 % d'UC obligatoire pour accéder au fonds euros
  - Maximum 50 000 EUR par adhérent sur le fonds euros
- **Note fonds euros** : 4,25/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-26 -->

- **Versement initial minimum** : 500 EUR
- **Versements libres minimum** : 50 EUR
- **Versements programmés minimum** : 50 EUR / mois ; 150 EUR / trimestre ; 300 EUR / semestre ; 600 EUR / an
- **Arbitrage minimum par support** : 10 EUR
- **Rachat partiel minimum** : 100 EUR
- **Règles d'arbitrage** : 0 % de frais ; options d'arbitrage automatique disponibles gratuitement
- **Limites d'investissement** :
  - Minimum 30 % en UC obligatoire (condition d'accès au fonds euros)
  - Maximum 50 000 EUR sur le fonds euros par adhérent
- **Conditions gestion déléguée** : gestion professionnelle par SwissLife Banque Privée ; surcoût +0,40 % / an
- **Avance sur contrat** : jusqu'à 80 % de l'épargne en fonds euros ; taux 4,54 % (2026)
- **Clause bénéficiaire** : Non communiqué [à vérifier]
- **Souscription non disponible pour** : mineurs, majeurs protégés, co-souscripteurs, personnes de plus de 85 ans, non-résidents français

## Fiscalité
<!-- Mis à jour : 2026-05-26 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)
