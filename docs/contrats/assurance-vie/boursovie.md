# BoursoVie

> Assurance vie distribuée par BoursoBank, assurée par Generali Vie (groupe Generali).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Distributeur** : BoursoBank (banque en ligne, filiale de Société Générale)
- **Assureur** : Generali Vie (groupe Generali)
- **Type** : Multisupports, contrat groupé
- **Modes de gestion** : Libre, pilotée (Edmond de Rothschild AM et Sycomore AM)
- **Disponibilité** : En ligne exclusivement (nécessite un compte BoursoBank)
- **Versement minimum initial** : 300 EUR
- **Année de création** : 1995

## Frais
<!-- Mis à jour : 2026-05-26 -->

- **Versement** : 0 %
- **Gestion UC** : 0,75 % / an maximum
- **Gestion fonds euros** : 0,75 % / an maximum (Eurocroissance : 0,75 % / an)
- **Arbitrage** : 0 %
- **Frais de performance** : 15 % sur le fonds croissance Générations Croissance Durable (source : sinvestir)
- **Surcoût gestion pilotée** : 0 € de frais supplémentaires sur la gestion pilotée (seuls les frais des fonds sélectionnés s'appliquent)
- **Frais de rachat** : 0 %
- **Autres frais** : 0 € sur les versements programmés, 0 € sur le changement de mandat en gestion pilotée
- **Note frais** : 4,5/5 (source : sinvestir)

## Fonds euros
<!-- Mis à jour : 2026-05-26 -->

- **Euro Exclusif** : garanti à 100 % — 100 % possible en gestion libre
  - Performance 2024 : 3,00 % à 3,50 % (source : sinvestir)
  - Performance 2025 : 3,00 % net (source : BoursoBank)
  - Performance cumulée 5 ans : 12,75 % (source : sinvestir)
  - Offre promotionnelle 2026 : objectif 4,50 % net (non garanti) avec bonus +1,50 % pour versements réalisés du 01/01/2026 au 30/06/2026 avec au moins 30 % d'UC (plafond 500 000 EUR) — source : BoursoBank
  - Fonds exposé en partie sur des supports immobiliers
- **Eurossima** (fonds euro, fermé aux nouveaux investissements) :
  - Performance 2024 : 1,70 % (source : sinvestir)
  - Performance 2025 : 1,67 % (source : sinvestir)
  - Performance cumulée 5 ans : 7,42 % (source : sinvestir)
- **Générations Croissance Durable** (fonds croissance, garantie partielle) :
  - Garantie de 80 % du capital net investi à l'échéance (horizon minimum 8 ans)
  - Performance 2025 : 3,40 % net (01/01/2025–31/12/2025) — source : BoursoBank
  - Composition : obligations d'entreprise, obligations d'État, actions, cash
  - Intègre des critères ESG, soutient le financement de PME
  - Frais de performance : 15 % (source : sinvestir)
  - Centralisation le vendredi, arbitrage en J+1
- **Contraintes d'accès** : le fonds Eurossima est fermé à la souscription pour les nouveaux investissements
- **Note fonds euros** : 4,5/5 (source : sinvestir)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-26 -->

- **Versement initial minimum** : 300 EUR (source : BoursoBank)
- **Versements libres minimum** : 25 EUR (source : BoursoBank — pages UC et supports immobiliers)
- **Versements programmés minimum** : 50 EUR / mois (source : BoursoBank — page Générations Croissance Durable)
- **Arbitrage minimum** : Non communiqué [à vérifier]
- **Règles d'arbitrage** : 0 % de frais — exécution en J+1 (source : BoursoBank)
- **Limites d'investissement** :
  - SCPI : maximum 200 000 EUR par produit (source : BoursoBank)
  - SCI / SC Terres Invest : maximum 30 % de l'encours du contrat, toutes SCI confondues (source : BoursoBank)
  - FCPR / private equity : capital bloqué sur la durée du fonds (sauf cas exceptionnels)
  - Offre bonus fonds euros 2026 : plafonnée à 500 000 EUR de versements, sous condition de 30 % minimum en UC (source : BoursoBank)
- **Conditions gestion pilotée** :
  - Edmond de Rothschild AM : 5 profils (Défensif, Équilibré, Réactif, Dynamique, Offensif)
  - Sycomore AM : 3 profils ISR (Défensif, Équilibré, Dynamique) — horizon minimum 3, 4 ou 6 ans selon profil
  - 0 € de frais supplémentaires de mandat — seuls les frais des fonds s'appliquent
  - 0 € pour changement de mandat (source : BoursoBank)
- **Conditions d'accès** : un compte BoursoBank est obligatoire pour souscrire
- **Rachat partiel** : disponible en 72h (rachat partiel en ligne) — source : BoursoBank
- **Avance sur contrat** : Non communiqué [à vérifier]
- **Clause bénéficiaire** : Non communiqué [à vérifier]

## Fiscalité
<!-- Mis à jour : 2026-05-26 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)
- **Imposition** : seules les plus-values sont imposées lors d'un retrait — le capital reste non imposable

## Avis synthétique
<!-- Mis à jour : 2026-05-26 -->

- **Note globale** : 4,5/5 (source : sinvestir)
- **Points forts** :
  - Aucun frais d'entrée, d'arbitrage ni de rachat
  - Fonds Euro Exclusif performant : 3 % net en 2025 (objectif 4,50 % avec bonus 2026)
  - Gestion pilotée sans surcoût de mandat (Edmond de Rothschild AM, Sycomore AM ISR)
  - Très vaste choix d'UC : +600 supports dont ETF, titres vifs, private equity, terres agricoles, produits structurés garantis à l'échéance
  - Versements libres dès 25 EUR
  - Rachat partiel disponible en 72h
  - Ergonomie reconnue et application mobile complète
  - Services intégrés d'une banque en ligne complète
- **Points faibles** :
  - Frais de gestion annuels de 0,75 % (vs 0,50 % chez certains concurrents comme Linxea Spirit 2)
  - Frais de performance de 15 % sur le fonds Générations Croissance Durable
  - Fonds Eurossima fermé à la souscription et peu performant (1,67 % en 2025)
  - Seulement 2 SCPI ouvertes à la souscription (taux de redistribution 90 % seulement)
  - Nombre d'ETF limité (40+) vs concurrents spécialisés
  - Compte BoursoBank obligatoire pour souscrire
- **Services complémentaires** : tous les services d'une banque en ligne (compte courant, bourse, PEA, PER, crédit immobilier, etc.) — source : sinvestir
