# Lucya CNP

> Assurance vie distribuée par Lucya, assurée par CNP Assurances (filiale de La Banque Postale).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Lucya (~170 000 clients, ~6 Md€ d'encours)
- **Assureur** : CNP Assurances (filiale de La Banque Postale — solvabilité : Non communiqué)
- **Type** : Multisupports individuel
- **Modes de gestion** : Libre, pilotée (3 profils)
- **Disponibilité** : En ligne exclusivement
- **Versement minimum** : 500 EUR
- **Année de création** : 2001

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,30 % / an
- **Gestion fonds euros** : 0,70 % / an
- **Arbitrage** : 0 % (illimité et gratuit)
- **Surcoût gestion pilotée** : 0,25 % / an
- **Frais de rachat** : 0 %
- **Autres frais** :
  - 0,07 % de frais de transaction sur les titres vifs
  - 3 % de frais sur les versements de rente
  - 5 % maximum de frais de rachat pour les supports en unités de compte [à vérifier — contradiction avec « 0 % de frais de rachat » affiché sur le site]
  - 2 % maximum de frais d'entrée spécifiques à certains supports UC (dispositions particulières)
- **Note frais** : 4,64/5 (source : sinvestir.fr)

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponibles** : CNP Lucya Euros B (garantie permanente du capital hors frais de gestion — 100 % possible)
- **Performance 2024** : 2,38 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance 2025** : 2,33 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance cumulée 3 ans (2023+2024+2025)** : 6,99 %
- **Contraintes d'accès** : aucune contrainte — investissement à 100 % possible
- **Note fonds euros** : 3/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 500 EUR
- **Versements libres minimum** : 500 EUR (source : sinvestir.fr)
- **Versements programmés minimum** : 50 EUR / mois, 150 EUR / trimestre, 300 EUR / semestre, 600 EUR / an
- **Arbitrage minimum** : 20 EUR (résidu minimum par support cédé : 20 EUR)
- **Arbitrage** : délai d'exécution J+1 (jour ouvré suivant)
- **Règles d'arbitrage** : arbitrages illimités et gratuits ; aucune contrainte particulière mentionnée sur le fonds euros
- **Limites d'investissement** : 100 % fonds euros possible ; limites SCPI/immobilier Non communiqué [à vérifier]
- **Conditions gestion pilotée** :
  - 3 profils : modérée, équilibrée, dynamique
  - Surcoût : 0,25 % / an sur les UC
  - Changement de profil ou de mode de gestion : gratuit, à tout moment
- **Rachat partiel minimum** : 150 EUR
- **Avance sur contrat** :
  - Minimum : 8 000 EUR
  - Maximum : 50 % du capital en UC ou 80 % du capital en euros (à la date de la demande)
  - Durée maximale : 3 ans
  - Conditions : soumis à l'accord de l'assureur ; intérêts applicables
- **Clause bénéficiaire** : Non communiqué [à vérifier]

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 4,5/5 (source : sinvestir.fr)
- **Points forts** :
  - Frais de gestion UC très bas (0,30 % / an — parmi les plus bas du marché)
  - 0 % de frais sur versements et arbitrages
  - Accès 100 % fonds euros sans contrainte
  - Plus de 600 titres vifs à frais réduits (0,07 %)
  - Avance sur contrat disponible (min 8 000 EUR, max 3 ans)
  - Assureur solide : CNP Assurances (filiale de La Banque Postale)
  - 4 options de gestion automatique gratuites
- **Points faibles** :
  - Fonds euros peu performant (2,33 % en 2025 vs 3 % + chez certains concurrents)
  - Seulement 15 ETF disponibles (vs 72 chez Lucya Cardif ou 80 chez Linxea Spirit 2)
  - Pas d'application mobile
  - 5 % de frais de rachat potentiels sur certains supports UC [à éclaircir]
  - Nombre de SCPI limité (10 selon sinvestir.fr)
- **Services complémentaires** : PER, PEA assurantiel, contrat de capitalisation (Lucya CNP Capi), SCPI
