# Corum Life

> Assurance vie distribuée par Corum L'Epargne, assurée par Corum Life (compagnie d'assurance du groupe Corum).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Corum L'Epargne (152 000 épargnants, +9,6 Md€ d'actifs sous gestion groupe)
- **Assureur** : Corum Life — compagnie d'assurance filiale du groupe Corum (solvabilité : rapport SFCR 2025 disponible sur corum.fr)
- **Type** : Multisupports individuel
- **Modes de gestion** : Libre (à la carte), gestion profilée (6 formules), mixte (libre + profilée)
- **Disponibilité** : En ligne exclusivement
- **Versement minimum** : 50 EUR
- **Année de création** : 2011

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0 % (frais de gestion inclus dans les performances nettes annoncées des fonds)
- **Gestion fonds euros** : 0,60 % / an (Corum EuroLife)
- **Arbitrage** : 0 %
- **Surcoût gestion profilée** : 0 % (formules incluses sans surcoût de contrat)
- **Frais de rachat** : 0 %
- **Frais de souscription sur les fonds** : compris dans les fonds — SCPI : jusqu'à 12 % à la souscription (assimilable à ~1,1 %/an sur 10 ans) ; fonds obligataires : jusqu'à 5 % — intégrés dans les DIC de chaque support
- **Frais de gestion des UC** : entre 0,25 % et 1,7 % / an selon le fonds (inclus dans les performances nettes affichées)
- **Note frais** : 5/5

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Fonds disponible** : Corum EuroLife (garanti à 100 % net de frais de gestion — 25 % maximum du contrat)
- **Performance 2024** : 4,65 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance 2025** : 4,10 % (net de frais de gestion, hors prélèvements fiscaux et sociaux)
- **Performance cumulée 2023+2024+2025** : 13,20 %
- **Contraintes d'accès** : 25 % maximum de l'épargne du contrat — pas d'investissement à 100 % en fonds euros possible
- **Note fonds euros** : 4,5/5

## Unités de compte
<!-- Mis à jour : 2026-05-16 -->

- **Nombre total de supports** : 10
- **ETF** : 0
- **SCPI** : 3 ([Corum Origin](../../actifs/scpi/corum-origin.md), [Corum XL](../../actifs/scpi/corum-xl.md), [Corum Eurion](../../actifs/scpi/corum-eurion.md)) — limite à 55 % du contrat
- **Fonds obligataires** : 4 (BCO, Corum BEHY, Corum Visio, Corum Tellia)
- **Fonds ISR / verts** : 2 (Sienna Solidaire, Sienna Obligations Vertes — gérés par Sienna Gestion)
- **Titres vifs** : non
- **Rééquilibrage automatique** : non
- **Formule Rosetta** : fonds mixte spécifique (20 % immobilier / 80 % fonds mixte) — performance 2025 : 7,11 %
- **SCPI : redistribution des loyers** : 100 % des loyers bruts versés
- **Note UC** : 3,5/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 50 EUR
- **Versements libres minimum** : 50 EUR [à vérifier]
- **Versements programmés minimum** : Non communiqué
- **Arbitrage minimum** : Non communiqué
- **Règles d'arbitrage** : 0 % de frais d'arbitrage
- **Limites d'investissement** :
  - SCPI : 55 % maximum du contrat (toutes SCPI confondues)
  - Fonds obligataires : 100 % possible
  - Fonds euros (Corum EuroLife) : 25 % maximum — investissement à 100 % en fonds euros impossible
- **Conditions gestion profilée** : 4 formules prédéfinies (Essentiel, Entreprises, Immo, Rosetta) + gestion mixte (combinaison libre + profilée) possible — 0 % de surcoût
- **Garantie décès** : 100 % des sommes investies en SCPI, obligataire et fonds mixte garantis jusqu'à 65 ans ; diminue de 5 % par an entre 65 et 75 ans — gratuite
- **Rachat partiel minimum** : Non communiqué [à vérifier]
- **Avance sur contrat** : Non communiqué [à vérifier]
- **Clause bénéficiaire** : standard / libre [à vérifier]
- **Délai de jouissance SCPI** : 5 mois (identique à l'achat en direct)

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 4,5/5 (source : sinvestir.fr)
- **Points forts** :
  - Accessible dès 50 EUR — l'un des seuils les plus bas du marché
  - 0 % de frais sur le contrat (versement, arbitrage, rachat)
  - Fonds euros très performant : 4,10 % en 2025, 13,20 % cumulé sur 3 ans
  - Accès exclusif aux SCPI Corum en assurance vie (Origin, XL, Eurion)
  - Fonds obligataires ISR inclus (Sienna Solidaire, Sienna Obligations Vertes)
  - Garantie décès 100 % gratuite jusqu'à 65 ans
  - Application mobile disponible
  - Gestion profilée sans surcoût (4 formules)
- **Points faibles** :
  - Univers d'investissement très limité : uniquement fonds maison Corum — aucun ETF, aucune diversification externe
  - Fonds euros limité à 25 % — impossible d'être 100 % sécurisé
  - SCPI limitées à 55 % et délai de jouissance de 5 mois
  - Nécessite une seconde assurance vie pour diversifier
  - Pas de titres vifs, pas de gestion pilotée externe
- **Services complémentaires** : SCPI en direct, PER (Corum PERLife), contrat de capitalisation (Corum Capi)
