# Yomoni Vie

> Assurance vie distribuée par Yomoni, assurée par Suravenir (filiale du Crédit Mutuel Arkea).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Yomoni (robo-advisor français, 80 000+ clients, 2+ Md€ sous gestion, certifié B Corp)
- **Assureur** : Suravenir (filiale du Crédit Mutuel Arkea, noté A+ par S&P)
- **Type** : Multisupports
- **Modes de gestion** : Pilotée (défaut, 1 000 EUR minimum), libre (50 000 EUR minimum)
- **Disponibilité** : En ligne exclusivement
- **Versement minimum** : 1 000 EUR (gestion pilotée) / 50 000 EUR (gestion libre)
- **Année de création** : 2015

## Frais
<!-- Mis à jour : 2026-05-16 -->

- **Versement** : 0 %
- **Gestion UC** : 0,60 % / an
- **Gestion fonds euros** : 0,60 % / an (complément possible jusqu'à 3 % max en cas de situation financière exceptionnelle)
- **Arbitrage** : 0 %
- **Surcoût gestion pilotée** : +0,70 % / an (soit 1,30 % total hors frais ETF internes)
- **Frais ETF internes** : environ 0,30 % / an (soit 1,60 % total all-in en gestion pilotée)
- **Frais de rachat** : 0 %
- **Frais de rente** : 3 % sur les versements de rente
- **Note frais** : 4,5/5

## Fonds euros
<!-- Mis à jour : 2026-05-16 -->

- **Suravenir Rendement 2** : garanti à 99,4 % — 80 % maximum du contrat
  - Performance 2024 : 2,20 %
  - Performance 2025 : 2,10 %
  - Performance cumulée 5 ans : 9,6 %
- **Suravenir Opportunités 2** : garanti à 97 % — 70 % maximum — disponible lors de certaines périodes uniquement, non accessible en continu
  - Performance 2024 : 2,50 %

- **Contraintes d'accès** :
  - Suravenir Rendement 2 : 80 % maximum du contrat
  - Suravenir Opportunités 2 : inéligible aux arbitrages ; proposé lors de certaines périodes seulement ; 70 % maximum
- **Note fonds euros** : 4,5/5

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 1 000 EUR (gestion pilotée) / 50 000 EUR (gestion libre)
- **Versements libres minimum** : Non communiqué [à vérifier]
- **Versements programmés minimum** : Non communiqué [à vérifier] ; automatisation disponible via l'interface
- **Arbitrage minimum** : Non communiqué [à vérifier]
- **Règles d'arbitrage** :
  - Suravenir Opportunités 2 : inéligible aux arbitrages — accessible uniquement par versement lors des périodes d'ouverture
  - Gestion pilotée : arbitrages réalisés par l'équipe Yomoni via mandat d'arbitrage ; le client ne sélectionne pas lui-même les supports
- **Limites d'investissement** :
  - Suravenir Rendement 2 : 80 % maximum du contrat
  - Suravenir Opportunités 2 : 70 % maximum, disponible par périodes uniquement
  - Pas de SCPI disponibles
- **Gestion pilotée** :
  - 10 profils de risque numérotés (Profil 2 Prudent à Profil 10 Audacieux C)
  - Versions ESG disponibles pour tous les profils
  - Gestion 100 % ETF / Multi-actifs selon les profils
  - Impossibilité de répartir un contrat sur plusieurs profils simultanément
  - Frais tout compris : 1,30 % / an (hors frais internes ETF ≈ 0,30 %) soit ~1,60 % all-in
- **Gestion libre** : accessible à partir de 50 000 EUR ; sélection autonome parmi les 203 UC disponibles
- **Multi-compartiments** : possibilité d'organiser plusieurs projets au sein d'un seul contrat
- **Rachat partiel minimum** : Non communiqué [à vérifier] ; rachats disponibles à tout moment sans pénalité
- **Avance sur contrat** : Non communiqué [à vérifier]
- **Clause bénéficiaire** : standard / libre (100 % en ligne)
- **Assurance-vie enfant** : disponible

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)
- **Protection** : Fonds de Garantie des Assurances de Personnes (FGAP) jusqu'à 70 000 EUR par bénéficiaire par contrat
