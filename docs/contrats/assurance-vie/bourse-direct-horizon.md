# Bourse Direct Horizon

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Distributeur** : Bourse Direct (courtier en ligne, ORIAS n° 08044344)
- **Assureur** : Generali Vie
- **Type** : Multisupports (contrat groupé, adhésion individuelle facultative)
- **Modes de gestion** : Libre, pilotée (mandatée par DNCA)
- **Disponibilité** : En ligne et en agence (4 agences en France)
- **Versement minimum** : 300 EUR
- **Année de création** : 1996

<!-- Mis à jour : 2026-05-26 -->

- **Versement** : 0 %
- **Gestion UC** : 0,65 % / an (gestion libre) ; 0,85 % / an (gestion pilotée)

- **Arbitrage** : 0 %
