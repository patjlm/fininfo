# Garance Épargne

> Assurance vie distribuée et assurée par Garance (mutuelle soumise aux dispositions du Livre II du Code de la mutualité).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-16 -->

- **Distributeur** : Garance (mutuelle soumise aux dispositions du Livre II du Code de la mutualité — SIREN 391 399 227 — siège : 51, Rue de Châteaudun, 75442 Paris Cedex 09)
- **Assureur** : Garance (solvabilité : 249,8 % — ratio SCR 2024, source : Good Value for Money d'après rapports SFCR Garance)
- **Type** : Multisupports (fonds euros + UC)
- **Modes de gestion** : Libre, pilotée (4 profils : Prudent, Équilibré, Dynamique, Audacieux), gestion à horizon
- **Disponibilité** : En ligne
- **Versement minimum** : 50 EUR (versement initial) ; 20 EUR/mois (versements programmés)
- **Année de création** : 1986

- **Versement** : 0 % (depuis le 1er janvier 2026 — précédemment 1 %)
- **Gestion UC** : 0,70 % / an (source : sinvestir.fr — site officiel garance.com inaccessible au moment de la vérification)
- **Gestion fonds euros** : 0,60 % / an
- **Arbitrage** : 0 %
- **Changement de mode de gestion** : 0 %
- **Surcoût gestion pilotée** : +0,10 % / an sur la part en UC (hors profil Sécurité)
- **Frais de rachat** : 0 %
- **Autres frais** : 3 % sur les versements de rente
- **Note frais** : 5/5 (source : sinvestir.fr)

## Fonds euros

- **Performance 2025** : 3,50 % (net de frais de gestion, brut de prélèvements sociaux et fiscaux)

- **Contraintes d'accès** : aucune — 100 % fonds euros possible sans quota d'UC obligatoire
- **Note fonds euros** : 5/5 (source : sinvestir.fr)

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-16 -->

- **Versement initial minimum** : 50 EUR
- **Versements libres minimum** : 150 EUR
- **Versements programmés minimum** : 20 EUR / mois
- **Arbitrage minimum** : Non communiqué [à vérifier]
- **Règles d'arbitrage** : 0 % de frais ; libre entre tous les supports
- **Limites d'investissement** : 100 % fonds euros possible sans contrainte de quota UC
- **Solde minimum après rachat partiel** : 800 EUR (source : Good Value for Money, analyse conditions générales 2025)
- **Conditions gestion pilotée** : surcoût +0,10 % / an sur la part UC (hors profil Sécurité) ; 4 profils disponibles : Prudent, Équilibré, Dynamique, Audacieux
- **Conditions gestion à horizon** : allocation auto-ajustée selon la durée restante avant l'échéance choisie
- **Rachat partiel minimum** : Non communiqué [à vérifier]
- **Avance sur contrat** : Non communiqué [à vérifier]
- **Clause bénéficiaire** : standard / libre
- **Offre de bienvenue (jusqu'au 31/08/2026)** : abondement jusqu'à 1 000 EUR pour versement initial ≥ 5 000 EUR avec 30 % minimum en UC (gestion libre) et absence de rachat pendant 6 mois ; limité à 2 contrats par personne

## Fiscalité
<!-- Mis à jour : 2026-05-16 -->

- **Avant 8 ans** : PFU 30 % (12,8 % IR + 17,2 % PS) ou barème IR + PS
- **Après 8 ans** : abattement annuel 4 600 EUR (célibataire) / 9 200 EUR (couple) puis 24,7 % (7,5 % + 17,2 % PS) ou barème IR + PS
- **Transmission** : abattement 152 500 EUR / bénéficiaire (versements avant 70 ans) ; 30 500 EUR tous bénéficiaires confondus (versements après 70 ans)

## Avis synthétique
<!-- Mis à jour : 2026-05-16 -->

- **Note globale** : 4,2/5 (source : sinvestir.fr)
- **Points forts** :
  - Fonds euros très performant (3,50 % net en 2024 et 2025, 3e année consécutive, bien au-dessus de la moyenne du marché)
  - Fonds euros accessible à 100 % sans quota UC obligatoire
  - Frais très compétitifs : 0 % versement (depuis 01/01/2026), 0 % arbitrage, 0 % rachat
  - Accessible dès 50 EUR, versements programmés dès 20 EUR/mois
  - Mutuelle indépendante (pas de rétrocommission sur les UC)
  - Solvabilité élevée (249,8 % en 2024)
- **Points faibles** :
  - Univers UC limité (46 supports, 1 seule SCPI)
  - Pas d'application mobile
  - Service client aux avis mitigés
  - Site officiel (garance.com) inaccessible au moment de la rédaction (erreur SSL)
- **Services complémentaires** : PER (Garance Sérénité, Garance Vivacité), mutuelle santé, prévoyance
