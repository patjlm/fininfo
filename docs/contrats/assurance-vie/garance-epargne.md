# Garance Épargne

> Assurance vie distribuée et assurée par Garance (mutuelle soumise aux dispositions du Livre II du Code de la mutualité).

Voir aussi : [Enveloppe Assurance Vie](../../enveloppes/assurance-vie.md) pour les règles générales.

## Informations générales
<!-- Mis à jour : 2026-05-26 -->

- **Distributeur** : Garance (mutuelle soumise aux dispositions du Livre II du Code de la mutualité — SIREN 391 399 227 — siège : 51, Rue de Châteaudun, 75442 Paris Cedex 09)
- **Assureur** : Garance (solvabilité : 249,8 % — ratio SCR 2024, source : [ACPR bulletin n°173](https://acpr.banque-france.fr/fr/publications-et-statistiques/publications/ndeg-173-la-situation-des-assureurs-soumis-solvabilite-ii-en-france-fin-2024))
- **Type** : Multisupports (fonds euros + UC)
- **Modes de gestion** : 3 options (source : [DIC Garance Épargne, 10/04/2025](https://www.garance.com/app/uploads/2025/06/KID_GARANCE_EPARGNE_GENERIC_FR_EUR_20250410_113237.pdf)) :
  - Option 1 — Gestion sécurité : 100 % fonds euros
  - Option 2 — Gestion profilée (Garance Smart Life) : 4 profils (Prudence, Équilibre, Dynamisme, Audace) × 3 orientations (Classique, ISR, Passive)
  - Option 3 — Gestion libre : allocation libre entre tous les supports
- **Disponibilité** : En ligne
- **Versement minimum** : 50 EUR (versement initial) ; 20 EUR/mois (versements programmés)
- **Année de création** : 1986

## Conditions et contraintes spécifiques
<!-- Mis à jour : 2026-05-26 -->

Source : [DIC Garance Épargne gestion libre, 10/04/2025](https://www.garance.com/app/uploads/2025/06/KID_GARANCE_EPARGNE_Gestion_libre_FR_EUR_20250410_113311.pdf) et [page produit officielle](https://www.garance.com/offres/assurance-vie-garance-epargne)

- **Limites d'investissement** : 100 % fonds euros possible sans contrainte de quota UC

- **Rachat partiel minimum** : Non communiqué [à vérifier]
- **Avance sur contrat** : Non communiqué [à vérifier]

- **Garantie décès accidentel** : doublement du capital (max 50 000 EUR) pour les adhérents < 60 ans en gestion profilée (hors profil Sécurité et gestion libre)
- **Clause bénéficiaire** : standard / libre
- **Offre de bienvenue (jusqu'au 31/08/2026)** : prime selon versement initial avec minimum 30 % en UC :

## Avis synthétique
<!-- Mis à jour : 2026-05-26 -->

- **Note globale** : 4,55/5 (source : sinvestir.fr)
- **Points forts** :
  - Fonds euros très performant (3,50 % net en 2024 et 2025, 3e année consécutive, bien au-dessus de la moyenne du marché)
  - Fonds euros accessible à 100 % sans quota UC obligatoire
  - Frais très compétitifs : 0 % versement, 0 % arbitrage, 0 % rachat
  - Accessible dès 50 EUR, versements programmés dès 20 EUR/mois
  - Mutuelle indépendante (sans but lucratif, pas de rétrocommission sur les UC)
  - Solvabilité élevée (249,8 % SCR en 2024)
  - Garantie décès accidentel doublement du capital (max 50 000 EUR) pour les profils profilés < 60 ans
- **Points faibles** :
  - Univers UC limité (~46 supports dont 19 ETF, 1 seule SCPI)
  - Pas d'application mobile
  - Service client aux avis clients mitigés (3/5 malgré note SAV de 4,25/5)
- **Services complémentaires** : PER (Garance Vivacité, Garance Sérénité), mutuelle santé, prévoyance
