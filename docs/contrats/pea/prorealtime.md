# ProRealTime — PEA

> PEA proposé par [ProRealTime](https://www.prorealtime.com/fr/pea) (France), avec tenue de compte et exécution des ordres assurées par Interactive Brokers (Irlande). Voir aussi la fiche [PEA (enveloppe)](../../enveloppes/pea.md) pour les règles générales.

## Frais de courtage
<!-- Mis à jour : 2026-05-16 -->

- **Grille tarifaire** (3 formules, actions/ETF France) :
  - **Tarif Standard** : 0,095 % de la valeur de la transaction (min. 4,95 EUR / max. 49 EUR)
  - **Tarif Actif** (> 15 ordres/mois en moyenne) : 0,08 % (min. 3,95 EUR / max. 49 EUR)
  - **Tarif Trader** (> 30 ordres/mois en moyenne) : 0,06 % (min. 2,95 EUR / max. 49 EUR)
- **Actions hors France (EUR)** : à partir de 0,06 % (min. 2,95 EUR)
- **Droits de garde** : 0 EUR
- **Frais de transfert** : 0 EUR (hors éventuels frais facturés par l'ancien courtier)
- **Frais de change** : aucun (PEA limité aux titres en EUR)
- **Frais divers** : aucun
- **Plafond réglementaire** : frais plafonnés à 0,5 % du montant de l'ordre (loi PACTE)
- **Exemples concrets (tarif Trader)** :
  - Ordre de 200 EUR : 1 EUR
  - Ordre de 2 000 EUR : 2,95 EUR
  - Ordre de 5 000 EUR : 3,73 EUR
  - Ordre de 10 000 EUR : 7,20 EUR
  - 3 ordres de 100 EUR/mois : 3 x 0,50 EUR = 1,50 EUR (plafond loi PACTE : 0,50 % appliqué car < minimum)
  - 3 ordres de 400 EUR/mois : 3 x 0,50 % x 400 EUR = 6 EUR (plafond loi PACTE appliqué)
- **Note frais de courtage** : 3,5/5
- **Note frais divers** : 0/5 (aucun frais divers)
