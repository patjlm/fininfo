# Fininfo

Wiki markdown des produits financiers français : enveloppes, contrats, actifs, institutions.

Voir [docs/README.md](docs/README.md) pour la documentation complète.

## Avertissement

- Les fiches sont générées par des agents IA à partir de sources officielles et cross-vérifiées avec des comparateurs. Elles peuvent contenir des erreurs ou des données obsolètes
- Les données marquées `[à vérifier]` n'ont pas été confirmées sur la source primaire
- Les performances passées ne préjugent pas des performances futures
- Ce wiki n'est pas un conseil en investissement
